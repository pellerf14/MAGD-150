#FinpellerQuadbaloons.zip
The first file, the _Quadbaloons file_ involves my experimentation with color by using baloons to overlap and merge color together. There are three transparent rectangular baloons which follow the RGB color scheme. The green is in the back, overlapped by the blue with the red on top. 

![quadbaloons](https://github.com/user-attachments/assets/20235a00-e728-4300-9c65-85f1561b17e7)

#pellerFLab3.zip
_pellerFLab_ involves the *mouseX/Y* and *pmouseX/Y* commands as well as randomly selected colors and diameter size. When the mouse is moved across the canvas, a repeated line of circles follows it. These circles can be generated from a wide range of colors and opacity values as well as circle size variation.

![circleColors](https://github.com/user-attachments/assets/a07457f9-2b1f-4759-9348-93ba042d72eb)

#pellerLab6.zip
This specific display of code was created by accident. My original intention was to animate a random array of objects moving from left to right of the canvas. Something happened while I was adjusting the dimensions of my bezier curves and the area the objects would be animated across. Instead of what I expected, I saw what looked like a line of needles moving from left to right while the viewer was watching them from an overhead angle. This was cooler to me so I kept it.
